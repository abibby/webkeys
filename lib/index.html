WebKeys

WebKeys lets you select links buttons and form elements quickly without needing to use the keyboard. You simply press the colon key and type the letters next to the element you want to select. It's running right now, give it a try.

Install

  • Install Tampermonkey
  • Open the Tampermonkey dashboard
  • Under utilities enter https://abibby.github.io/webkeys/webkeys.js into the Import from URL
  • Click install
